# Albion Online Lore

## လွန်ခဲ့သော နှစ်ပေါင်း ၁၈၀၀

ထိုအချိန်က Albion ကို ကြောက်မက်ဖွယ်ကောင်းသော နဂါးကြီးတွေက အုပ်ချုပ်ထားတယ်။ ထိုရက်စက်ကြမ်းကြုတ်သော နဂါးကြီးတွေရဲ့ လက်အောက်မှာ ကမ္ဘာဦးလူသားတွေက မှော်ပညာနဲ့ အသားကျလာပြီး ဒရူးဒ်များ ဖြစ်လာကြတယ်။ သူတို့တွေက Albion ရဲ့ giant တွေနဲ့ သဘောတူညီမှုယူပြီး လူသားမျိုးနွယ်စု အမျိုးမျိုးကို ဖွင့်ဖြိုးတိုးတက်စေတယ်။ Giant တွေ နဲ့ ဒရူးဒ်တွေ လက်တွဲပြီး နဂါးတွေကို အိပ်ပျော်စေမယ့် မှော်ပညာကို အတူတကွ လုပ်ဆောင်ခဲ့တယ်။ ပျက်စီးနေတဲ့ ကမ္ဘာကြီးဟာ ပြန်လည်ကောင်းမွန်လာပြီး giant တွေ နဲ့ လူသားတွေ အတူတကွ အိပ်ပျော်နေတဲ့ နဂါးကြီးတွေကို စောင့်ကြပ်ကြပြီး Albion's Keepers များဖြစ်လာကြတယ်။
